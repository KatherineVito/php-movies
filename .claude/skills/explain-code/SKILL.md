description: Explica código usando diagramas visuales y analogías. Úsalo cuando expliques cómo funciona el código, enseñes sobre un codebase o cuando el usuario pregunte "¿cómo funciona esto?"
---

Cuando expliques código, siempre incluye:

1. **Empieza con una analogía**: Compara el código con algo de la vida cotidiana
2. **Dibuja un diagrama**: Usa ASCII art para mostrar el flujo, estructura o relaciones
3. **Recorre el código**: Explica paso a paso qué sucede
4. **Resalta un error común**: ¿Cuál es un error o malentendido frecuente?

Mantén las explicaciones conversacionales. Para conceptos complejos, usa múltiples analogías.
