# Flujo

1. Detectar los archivos modificados.

2. Revisar si los cambios importantes ya estan documentados en `README.md`.

3. Si no estan documentados, actualizar `README.md` de forma breve y clara.

4. Si ya estan documentados, no modificar `README.md`.

## Reglas

- Solo documentar cambios importantes.
- Revisar siempre `README.md` antes de editarlo.
- Mantener la documentacion clara y concisa.
